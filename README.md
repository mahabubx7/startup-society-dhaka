### Usage

To run the project, open the `index.html` in your browser.


### Run tests

To run tests, run the following command:


Testing with Lint-checks:

```sh
  npm test
```

or, you can test linters for each tech
```sh
npm test:html
```
```sh
npm test:styles
```
```sh
npm test:js
```
